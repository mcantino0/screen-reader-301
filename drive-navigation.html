Lesson 2: Google Drive - Navigation and File Creation

Now that your essential settings are in place, let's learn how to navigate Google Drive and create new files.

Basic Navigation

While in Google Drive, you’ll spend most of your time in the Navigation and Items areas. The Navigation area lists the main Google Drive folders (My Drive, Shared with Me, Starred, etc.). The Items area lists all the files for your current folder. To move between these areas, you’ll use a two-step “go to” command (Alt + G).

Task Keyboard Command
Go to Navigation Region Alt + G, then N
Go to Items (list of files) Alt + G, then I
Go to Details (info about specific file) Alt + G, then D

If you want to open a particular folder or file, just press Enter.

Creating New Content

The "go to" commands will help you locate and open existing files, but what if you want to make something new?

You can add new content to Google Drive by using the “New” button in the top left corner of the screen. This button will let you upload new folders or files from your computer, and it will also let you create new folders or new documents inside of Google Drive. Use the New button to quickly create new Google Docs or Google Slides files. The files will automatically be added to the folder currently opened in your Items file area. To access the "New" button, press Alt + N, then use the arrows keys to move through the menu contents.

Task Keyboard Command
Upload/Create New folders or files Alt + N
