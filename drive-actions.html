Lesson 3: Google Drive - Actions, Search, and Keyboard Help

Actions Menu (A.K.A. Context menu, A.K.A. right-click)

The Actions menu (also called the context menu or right-click menu) is where you can really take control of your files in Google Drive. Need to rename, move, make a copy, share, or download a file? Then you need the Actions menu. To open the Actions menu, press Alt + A on a file or folder, and use your arrow keys to move through the menu of options. You can also open the Actions menu for the folder that you're currently in by pressing Alt + Shift + A.

Task Keyboard Command
Actions menu for item Alt + A
Actions menu for parent folder Alt + Shift + A

Search

Sometimes hunting through all your folders can be time consuming. To make finding your materials easier, use the Search field at the top of the screen. Just press / to enter the search field, and type in the name of the file or folder you're looking for. You can access the Advanced Search tool by pressing Tab from the Search field, and then pressing Enter or Spacebar. Use the Tab key to move through the Advanced Search options, which will let you narrow your search to specific file types, owners, dates, and more.

Task Keyboard Command
Search field /
Advanced Search /, followed by Tab and Enter

Keyboard Help

To see a full list of keyboard commands for Google Drive, use Ctrl + /. You can search for a specific command, or you can read through a list of all Google Drive commands.

Task Keyboard Command
Open Keyboard Shortcut list Ctrl + /
