Lesson 2: Google Slides - 3 Main Areas

Navigating the Slides Environment

Task Keyboard Command
Move to Film Strip Ctrl + Alt + Shift + F
Move to Canvas Ctrl + Alt + Shift + C
Move to Speaker Notes Ctrl + Alt + Shift + S

There are 3 key areas to be aware of when working in Google Slides: the Film Strip, the Canvas, and the Speaker Notes for each slide. Youâ€™ll be interacting with each of these areas repeatedly as you create your presentation, so you should get comfortable moving between them.

  • Film Strip: A vertical strip on the left side of the screen that shows small preview images of each slide in your presentation.
  • Canvas: The main slide area where you build the content for individual slides in your presentation.
  • Speaker Notes: A text box below the Canvas where an author can leave presentation notes. These notes will not be shown on screen during your presentation.
