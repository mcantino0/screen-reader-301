Lesson 4: Gmail - Writing Emails

You can start composing a new email from anywhere in Gmail by pressing C, as long as you're not currently in a text field. If you are in a text field, press Esc first to exit "Forms mode".

To reach the options along the bottom of the email window (Send, Attachments, Emojis, etc.), use the Tab key. To use the CC and BCC fields, press Ctrl + Shift + C for CC and Ctrl + Shift + B for BCC.

When you're reading emails, you can also use commands for replying and forwarding. Use R to Reply, A to Reply All, and F to Forward an email.

Finally, you can take advantage of Gmail's spell check by pressing Ctrl + ; (semicolon) and Ctrl + ' (apostrophe) to move to the previous or next spelling error.

Task Keyboard Command
Compose new message (from anywhere) C
Reply R
Reply All A
Forward F
Use CC field Ctrl + Shift + C
Use BCC field Ctrl + Shift + B
Move to next spelling or grammar error Ctrl + ' (apostrophe)
Move to previous spelling or grammar error Ctrl + ; (semicolon)
Send email Ctrl + Enter
