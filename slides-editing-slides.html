Lesson 5: Google Slides - Editing Slide Contents

Now that we know the basics, we can start adding content to the slides. You’ll be editing slide content from the Canvas area. Keep close track of when you are editing or selecting an item in a slide because this can change the behavior of some commands.

  • You can move between elements on your slide with the Tab key.
  • Press Enter to start editing a text box to add text content.
  • When you’re done editing, press Esc to stop editing.
  • To remove an element on your slide, tab to that element and press Delete.
  • To adjust the position of an element, tab to it and use the arrow keys to move it left/right/up/down.
  • To move to the previous or next slide, use Page Up/Page Down (or the up/down arrows if nothing is selected).
  • Add images, tables, and more from the Insert menu.
  • Check text formatting with the Google command (Ctrl + Alt + A, then Ctrl + Alt + F) or the JAWS command (JAWS + F).
